## Running Immigration's Tests Locally

### Copying folder

	- Pindahkan folder ke packages
	- Run 'composer update'
	- Run 'composer dump-autoload'
	- Run 'composer dump-autoload -o'

### Run Migrate

	- Run migrate dalam sub folder immigration/src/databases/migrations
